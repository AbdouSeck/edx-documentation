
========================================
Feedback and Hints in Common Problems
========================================

You can now add feedback, hints, or both to the following problem types.

* Checkbox
* Dropdown
* Multiple Choice
* Numerical Input
* Text Input
  
In Studio, you can use new templates to add sample problems with the feedback
and hint syntax.
